# SistemiReti
Per far funzionare il progetto, segui questi passaggi: prima di tutto, assicurati di avere Java 8 o superiore installato sul tuo computer. Puoi verificare la versione con il comando `java -version`. Successivamente, scarica o copia tutti i file `.java` forniti nel progetto in una cartella. Una volta fatto, apri un terminale o prompt dei comandi e naviga nella cartella dove hai copiato i file. Compila i file `.java` con il comando `javac *.java`. Dopo aver compilato i file, avvia il server con il comando `java ServerMain`. Il server si metterà in ascolto sulla porta 12345. A questo punto, apri una nuova finestra del terminale, naviga di nuovo nella cartella contenente i file compilati e avvia il client con il comando `java Client`. Il client si connetterà al server e potrai iniziare a inviare comandi per ottenere i dati sugli ospedali.
